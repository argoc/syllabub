COURSE DETAILS:

 

Description: An introduction to HyperText Markup Language for first time users. Learn to use HTML tags to create web pages in accordance with XHTML specifications. Create links, format text, create bulleted and numbered lists, insert images and background color/images, produce image maps, create forms, and understand multimedia possibilities.

Credits: 1

Class Date: Saturday, November 1, 2014

Class Time: 8:30 a.m.-5:30 p.m.
Class Location: TEC-212 (Bremerton Campus)

Prerequisite: Basic knowledge of Microsoft Windows 7. Competent keyboard skills. Basic understanding of the Canvas Learning Management System to take quizzes, download and view course files, and submit assignments.

Text Required: HTML and CSS, 6th ed., Brief, by Patrick Carey, 2012 Course Technology
The Campus Bookstore is not open on Saturday.

 

Core Abilities: CIS 114 meets the core abilities of Communication and Thinking.

 

Topics Covered: See Class Agenda below.

 

Course Requirements: Attendance is considered part of your grade for this on-ground course, and will be noted throughout the day.

 

Learning Outcomes:

  • Student will be able to demonstrate planning, designing, and constructing a simple web site.
  • Student will be able to demonstrate text formatting and creating document structure.
  • Student will be able to demonstrate the inclusion of images, background images, and background color.
  • Student will be able to demonstrate the construction and use of a web page form.

 

 

 

 

 

 

Course Outcomes: Upon completion of this course, the student should be able to:

1.      Explain what HTML and CSS are

2.      Explain the difference between elements and attributes

3.      Explain the purpose of doctype

4.      Explain the use of head elements & identify common head elements

5.      Explain the role of the div tag

6.      Explain the use of grouping & identify common grouping elements

7.      Explain how to incorporate style sheets

8.      Explain the use of text-level elements & identify common text-level elements

9.      Identify common character entity references

10.  Demonstrate how to create an HTML5 web page using CSS

11.  Explain what a homepage is

12.  Explain ways to structure a web site

13.  Explain the role of the nav tag

14.  Explain what hypertext is & identify common hypertext attributes

15.  Explain the difference between a relative and absolute path

16.  Explain the role of the id attribute

17.  Explain what client-side image mapping is

18.  Demonstrate how to use a background image or background color on a web page

19.  Explain what an URL is & identify its main parts

20.  Explain what a local file is

21.  Explain what metadata is & identify common metadata elements

22.  Demonstrate how to create and use a simple Web form

23.  Demonstrate how to create a simple Web site with appropriate navigation links and use of metadata.

 

 

 

 

Learning Modes:

 

This is a Web-Enhanced class incorporating classroom time in Bremerton and the online classroom in the Canvas Learning Management System. Learning will be delivered in lecture, demonstration, discussion, and projects. Assignments and quizzes/exams must be submitted through Canvas.

 

It is expected that the student will have a basic knowledge of the use of a computer and internet access. This course is not an introduction to basic computer literacy or basic internet literacy. A basic understanding of and manipulation of the Windows 7 interface is expected. In particular, students should know how to use Windows Explorer to manipulate (save, copy, delete, move, etc.) files, how to get access to the Internet, how to use your browser, how to fill out Web forms, how to use email (including attachments), and similar basic computer and Web skills. If you do not feel comfortable with these basic skills, it is recommended that you consider taking CIS 107 or CIS 150 prior to taking this class to build computer literacy. Please have an alternate plan in case you have issues with your normal Internet access and related software that might affect your ability to complete this class.
